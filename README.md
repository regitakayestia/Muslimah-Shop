# Muslimah-Shop
Website E-Commerce Muslimah OOP PHP

##fitur
-halaman beranda
-login
-register
-daftar produk
-keranjang
-dashboar user
-logut

## cara menjalankan
1. import file database sql ke phpmyadmin
2. jalankan xampp
3. letakkan folder di xampp/htdocs/muslimah
4. akses localhost:8080/muslimah

## teknologi
-PHP, Native OOP
-MYSQL
